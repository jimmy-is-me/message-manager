# v1.1.0 更新總覽

## 📅 更新日期
2026-01-17

## 🎯 主要更新

### 1. **電子信箱必填功能** ✅
- 首次打開對話框時，客戶**必須**輸入Email地址
- Email會儲存在localStorage，下次使用更方便
- 提交後顯示確認訊息，讓客戶知道資訊已收到

### 2. **LINE風格UI優化** 🎨

#### 視覺改進：
- ✨ **漸層標題列**：使用LINE綠色漸層效果
- ✨ **改良訊息氣泡**：更好的陰影和邊框設計
- ✨ **背景紋理**：聊天區域添加微妙的斜紋背景
- ✨ **按鈕升級**：更大、更圓潤的按鈕設計
- ✨ **表單美化**：輸入框有漸層背景和圖示標籤

#### 新增表單標籤：
- 📧 電子信箱
- 👤 您的姓名
- 📱 聯絡電話

### 3. **動畫效果** ⚡

新增多種流暢動畫：

| 元素 | 動畫效果 |
|------|---------|
| 訊息氣泡 | 彈跳進入動畫 |
| 發送按鈕 | 持續脈衝效果 |
| 輸入框 | 聚焦光暈動畫 |
| 對話按鈕 | 懸停放大效果 |
| 狀態指示燈 | 閃爍動畫 |
| 未讀徽章 | 跳動提醒 |
| 標題圖示 | 緩慢脈衝 |

### 4. **新增檔案**

- `public/css/animations.css` - 進階動畫效果庫
- `GITHUB-UPLOAD-STEPS.md` - GitHub上傳指南

## 📱 使用者體驗提升

1. **首次使用流程**：
   ```
   打開對話框 → 看到歡迎訊息 
   → 必須輸入Email（帶圖示標籤）
   → 可選填姓名/電話
   → 發送第一則訊息
   → 系統確認已收到資訊
   → 表單自動收起
   → 開始對話
   ```

2. **再次使用**：
   ```
   打開對話框 → Email已預填
   → 表單自動隱藏
   → 直接開始對話
   ```

3. **視覺回饋**：
   - 輸入時有聚焦光暈
   - 按鈕有脈衝動畫提示可點擊
   - 訊息發送有彈跳動畫
   - 線上狀態有閃爍指示
