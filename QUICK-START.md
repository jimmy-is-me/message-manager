# 快速開始指南

5分鐘內開始使用 LINE官方帳號訊息管理系統！

## ⚡ 三步驟快速設定

### 步驟 1：安裝插件（1分鐘）

1. 下載插件ZIP檔案
2. WordPress後台 → **插件** → **安裝插件** → **上傳插件**
3. 選擇ZIP檔案並安裝
4. 點擊**啟用插件**

✅ 完成！側邊選單會出現「LINE訊息」項目。

### 步驟 2：設定Discord通知（2分鐘）

#### 取得Webhook URL：

1. 開啟Discord → 選擇頻道 → 點擊齒輪圖示
2. 整合 → Webhooks → 新增Webhook
3. 複製Webhook URL

#### 在插件中設定：

1. WordPress後台 → **LINE訊息** → **設定**
2. 貼上Webhook URL
3. 勾選「啟用Discord通知」
4. 點擊**測試連接**（確認Discord收到測試訊息）
5. 點擊**儲存設定**

✅ 完成！現在有新訊息時會收到Discord通知。

### 步驟 3：測試對話功能（2分鐘）

1. **前台測試：**
   - 開啟網站（無痕模式）
   - 點擊右下角對話框按鈕
   - 填寫資訊並發送測試訊息

2. **檢查Discord：**
   - 確認收到新訊息通知

3. **後台回覆：**
   - WordPress後台 → **LINE訊息** → **所有對話**
   - 點擊進入對話
   - 輸入回覆並發送

4. **確認前台收到：**
   - 回到前台對話框
   - 等待3-5秒
   - 看到管理員回覆

✅ 完成！系統已經可以正常使用了！

## 🎨 選用：自訂外觀（額外2分鐘）

1. **LINE訊息** → **設定**
2. 修改以下項目：
   - 對話框標題：改成您想要的標題
   - 主題顏色：選擇符合網站風格的顏色
   - 對話框位置：選擇左下角或右下角
3. **儲存設定**

## 📌 使用短代碼

想在特定頁面顯示對話框？在頁面內容中加入：

```
[line_message_chat]
```

## 🎯 日常使用流程

### 管理員：

1. 收到Discord通知
2. 點擊通知中的連結
3. 直接進入後台對話頁面
4. 輸入回覆並發送

### 客戶：

1. 點擊網站上的對話框按鈕
2. 填寫基本資訊（僅首次）
3. 輸入問題
4. 收到管理員回覆

## ❓ 常見快速問答

**Q: 需要LINE官方帳號嗎？**
A: 不需要！這是獨立的客服系統。

**Q: 訊息是即時的嗎？**
A: 接近即時，前台每3秒自動檢查新回覆。

**Q: 可以有多個管理員嗎？**
A: 可以，所有有管理員權限的使用者都能回覆。

**Q: Discord通知是必須的嗎？**
A: 不是必須，但強烈建議使用以便即時收到通知。

**Q: 客戶需要註冊嗎？**
A: 不需要，訪客可以直接發送訊息。

## 🚀 進階技巧

### 技巧 1：自訂主題顏色

在佈景主題的「自訂CSS」中加入：

```css
:root {
    --lmm-primary-color: #your-color;
}
```

### 技巧 2：調整對話框大小

```css
.lmm-chat-window {
    width: 400px;
    height: 650px;
}
```

### 技巧 3：在特定頁面隱藏浮動按鈕

```css
.page-id-123 .lmm-chat-widget {
    display: none;
}
```
